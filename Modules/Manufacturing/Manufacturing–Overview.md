# **1. Production**

Covers all planning, execution, and shop-floor activities required to run manufacturing operations.

| Feature               | Overview Description                                                                          |
| ----------------------- | --------------------------------------------------------------------------------------------- |
| **Work Order (WO)**          | Main document used to manufacture an item. Defines quantity, routing, and required materials. |
| **Production Plan**     | Converts demand into a plan for production, ensuring capacity and material availability.      |
| **Job Card**            | Operation-level task sheet for workers to perform and record production work.                 |
| **Downtime Entry**      | Logs machine downtime for maintenance, breakdown, or idle reasons.                            |

---

# **2. Bill of Materials**

Manages the product structure (materials + operations) and shop-floor resource definitions.

| Feature                   | Overview Description                                                                 |
| --------------------------- | ------------------------------------------------------------------------------------ |
| **Bill of Materials (BOM)** | Defines components, quantities, scrap %, and routing steps needed to make a product. |
| **Workstation Type**        | Category of workstation (machine type, manpower type).                               |
| **Workstation**             | Physical work center/machine where operations take place.                            |
| **Plant Floor**             | Real-time shop-floor screen to track operations, time, and production progress.      |                       
| **Operation**               | Individual tasks performed during manufacturing (cutting, welding, assembly, etc.).  |
| **Routing**                 | Sequence of operations required to manufacture a product.                            |

---

# **3. Tools**

Utility tools for managing and maintaining BOMs.

| Tool Name               | Overview Description                                                                |
| ----------------------- | ----------------------------------------------------------------------------------- |
| **BOM Update Tool**     | Bulk update or modify multiple BOMs at once.                                        |
| **BOM Comparison Tool** | Compares two BOMs to identify differences in components, quantities, or operations. |
